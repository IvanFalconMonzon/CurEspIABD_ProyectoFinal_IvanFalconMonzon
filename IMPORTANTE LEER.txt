Dónde encontrar los archivos del proyecto:
Para que entiendas mejor cómo está organizado este proyecto, te explico la ubicación de los archivos clave:

El conjunto de datos (dataset):
El conjunto de datos completo de imágenes (conocido como "dataset") no está disponible en GitHub. La razón es que son muchísimas imágenes, lo que hace que el archivo sea demasiado grande, incluso después de intentar comprimirlo.

Por lo tanto, para acceder a todas las imágenes que se usaron en este proyecto, deberás ir a mi Google Drive. Allí encontrarás una carpeta llamada "dataset", que es donde está todo el material visual.

La aplicación web:
La aplicación web, es decir, el software que permite interactuar con el proyecto, se encuentra dentro de la carpeta llamada "HardVision". Esta carpeta contiene todos los archivos necesarios para ejecutar y usar la aplicación.
